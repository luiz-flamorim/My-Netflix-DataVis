My Netflix visualisation

The diagram below is an experiment to visualise my user Netflix data since 2012, where each circle represents one film or serie I have watched. You can explore it by clicking on the different circles, colours and buttons. All the information about the films and series I am using on this visualisation can be found on The Movie Database website.

Click on the buttons bellow to change the chart type.

This website has been developed for portfolio purposes only. The objective is to display different data visualisations using D3, API integration, and data treatment using Node.JS. Please refer to my Github page to check the code I used, credits and also for contacting me.
